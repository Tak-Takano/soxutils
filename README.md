# SoX Utilities
Python の SoX ライブラリを使った、ユーティリティ集

現在はディレクトリ内のファイルの一括変換機能のみ

# Installation
1. git clone する
2. `pip install (/path/to/repository)`

# Usage
```bash
soxdir -i /path/to/input_dir -o /path/to/output_dir -r 16000 -b 16
```
